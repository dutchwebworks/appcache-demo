Offline webapp met HTML5 Application Cache

Webbrowser support

Dit is een demo voor HTML5 ApplicationCache.

Wel

  • Firefox 3.5+
  • Chrome 5.0+
  • Safari 4.0+
  • Opera 10.6+
  • iOS Safari 2.1+
  • Android 2.0+

Niet

  • Internet Explorer (zelfs niet v10)
  • Mobile Internet Explorer
